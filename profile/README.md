# NuVibe

**NuVibe is a next-generation music distribution and growth platform for independent African artists.**

We help artists distribute their music globally, understand their royalties clearly, and grow their careers with modern tools built for today's music ecosystem.

---

## Mission

Empower independent artists with transparent distribution, actionable analytics, and tools that accelerate sustainable music careers.

---

## Core Platform Features

### Distribution
- Submit releases to Spotify, Apple Music, Boomplay, YouTube Music, Audiomack, and more
- Track delivery status in real time
- Control release metadata, artwork, and contributors

### Royalties & Wallet
- Clear royalty breakdown per store and territory
- Real-time earnings tracking
- Wallet system for payout requests
- Admin-verified payouts

### Artist Growth Tools
- Release performance analytics
- TikTok sound submission support
- Learning Hub for artist education
- Release coach and promotion tracking

### Admin Console
- Artist approval workflow
- Royalty import system
- Payout approval system
- Platform oversight tools

---

## Platform Architecture

Frontend:
- React
- Vite
- TailwindCSS
- React Router

Backend (in development):
- Node.js
- PostgreSQL
- REST API

Current development uses json-server to simulate production backend behavior.
